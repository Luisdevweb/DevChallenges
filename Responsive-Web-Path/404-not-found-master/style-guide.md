# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

White: hsl(0, 0%, 100%)[background],[button]

Gray 1: hsla(0, 0%, 20%, 1) [heading]
Gray 2: hsla(0, 0%, 31%, 1) [paragraph]
Gray 3: hsla(0, 0%, 74%, 1) [footer]
